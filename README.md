# Reddit AI Bot

## Overview
This bot generates and posts content to Reddit daily using Groq AI.

## Features
- Daily automated posting.
- AI-generated content using Groq API.
- Logging and error handling.

## Requirements
- Python 3.8+
- Reddit API credentials
- Groq API key
